## üöÄ Getting Started

### Week 1-2: Setup & First Project
1. **Setup environment**
   - Create Azure free account
   - Sign up for Databricks Community Edition
   - Install VS Code + Python
   - Setup GitHub

2. **First hands-on project**
   - Clone [nanoGPT](https://github.com/karpathy/nanoGPT)
   - Follow the tutorial
   - Train your first GPT model

3. **Start tracking**
   - Download CSV files to Excel
   - Mark Week 1-2 as "In Progress"

### Recommended Weekly Structure
- **Monday-Wednesday**: Theory & reading (5-7 hours)
- **Thursday-Friday**: Code tutorials from GitHub repos (5-7 hours)
- **Weekend**: Build mini-project or work on deliverable (5-6 hours)

### Study Tips
1. **Don't skip foundations** - Math and theory pay off
2. **Code along** - Type every example yourself
3. **Build in public** - Share progress on GitHub
4. **Join communities** - Hugging Face, Databricks forums
5. **One concept at a time** - Master before moving on
